Flip 📱

Flip your phone. Focus instantly.

Flip is a productivity application that helps users enter deep work mode with a simple physical action—placing their phone face down.

Instead of manually opening a focus app, starting a timer, or enabling Do Not Disturb, Flip automatically detects when the device is flipped and instantly starts a distraction-free focus session.

✨ Features
1. 📲 Face-down gesture detection
2. ⏱ Automatic focus session start
3. 🔕 Distraction-free mode
4. 📊 Focus statistics and session history
5. 🎯 Custom focus durations
6. 🌙 Minimal and clean user interface
7. 🚀 One-action productivity workflow
🛠 Tech Stack
1. Flutter
2. Dart
3. Device Sensors (Accelerometer & Proximity)
4. Local Storage (SQLite)
5. State Management (Provider)
6. Android Focus & Notification APIs
🎯 Problem

Most productivity apps require multiple steps before users can start focusing. This friction often causes procrastination.

💡 Solution

Flip removes that friction entirely.

Simply place your phone face down on the desk and Flip automatically starts a focus session, silences distractions, and helps you stay in deep work.
